# UC19 - Basemap Attribution Control

## Status
Implemented (2026-03-15)

## Goal
Provide one visible attribution control that reflects the active basemap and preserves required provider credit text.

## Scope Delivered
- Disabled default MapLibre attribution control (`attributionControl: false`).
- Added custom `MapAttributionControl` in `MapView`.
- Streets basemap attribution text:
  - `© OpenStreetMap contributors` (linked to OSM copyright page)
- Satellite basemap attribution text:
  - `Powered by Esri`
  - appended with dynamic provider metadata when available

## Implementation Notes
- New files:
  - `MapView/MapAttributionControl.tsx`
  - `MapView/mapAttributionText.ts`
  - `MapView/arcgisAttribution.ts`
- `arcgisAttribution.ts` fetches ArcGIS World Imagery metadata (`?f=pjson`) and reads provider text from `attribution` or `copyrightText`.
- Metadata failures are non-fatal:
  - map remains usable
  - attribution falls back to `Powered by Esri`
  - exception is captured for observability

## Non-Goals / Out Of Scope
- No provider-name hardcoding for ArcGIS metadata tail.
- No attribution persistence in canonical project state.

## Acceptance Criteria
- Attribution control is visible and updates with basemap mode.
- OSM mode always shows required fixed wording.
- Satellite mode always shows at least `Powered by Esri`.
- Provider metadata, when available, is displayed in satellite mode.

## Validation

  - verifies OSM fixed wording
  - verifies Esri-only fallback
  - verifies Esri + provider metadata text
